# 🏗️ Gerenciador de Obra

Um aplicativo PWA (Progressive Web App) para gerenciar gastos, materiais e progresso de sua obra de forma simples e eficiente!

---

## 📱 Como Acessar o App

### **Link Direto:**
```
https://ai.studio/apps/5823d939-8b21-4633-962c-2532928aa573
```

---

## 🚀 Como Instalar no Celular

### **Android:**
1. Abra o link acima no navegador Chrome
2. Toque nos **3 pontinhos** (⋮) no canto superior direito
3. Selecione **"Instalar app"** ou **"Adicionar à tela inicial"**
4. Confirme a instalação
5. ✅ Pronto! O app aparecerá na sua tela inicial com ícone laranja 🏗️

### **iPhone/iPad:**
1. Abra o link acima no navegador Safari
2. Toque no **botão de compartilhamento** (↑ caixa com seta)
3. Role para baixo e selecione **"Adicionar à tela inicial"**
4. Nomeie como **"Obra"** (ou outro nome)
5. Toque **"Adicionar"**
6. ✅ Pronto! O app aparecerá na sua tela inicial 🏗️

## ✨ Recursos do App

✅ **📊 Acompanhamento de Gastos** - Controle total de despesas
✅ **📦 Controle de Materiais** - Organize seus materiais e estoque
✅ **📈 Relatórios e Análises** - Visualize dados em gráficos
✅ **💰 Orçamentos Personalizados** - Crie orçamentos por projeto
✅ **📱 Funciona Offline** - Use mesmo sem internet
✅ **⚡ Rápido e Fácil** - Interface intuitiva e simples

---

## 🎯 Funcionalidades Principais

- **Dashboard** - Visão geral do projeto
- **Gerenciador de Gastos** - Registre todas as despesas
- **Controle de Materiais** - Acompanhe estoque
- **Relatórios** - Gere relatórios de desempenho
- **Sync Automático** - Seus dados na nuvem

---

## 💡 Dicas Úteis

### Primeiro Acesso
1. Instale o app no celular (siga as instruções acima)
2. Abra o app
3. Crie seu primeiro projeto
4. Comece a registrar gastos e materiais

### Salvando Dados
- Todos os dados são salvos automaticamente
- Você pode usar offline
- Ao conectar à internet novamente, os dados sincronizam

### Compartilhando Dados
- Você pode exportar relatórios
- Compartilhe com sua equipe
- Acompanhe o progresso em tempo real
