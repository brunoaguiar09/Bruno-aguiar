# 🏗️ Gerenciador de Obra - PWA

Um aplicativo web progressivo para gerenciar sua obra, controlar gastos, materiais e progresso de forma simples e eficiente.

## ✨ Recursos

- 📊 **Acompanhamento de Gastos** - Controle todos os gastos da sua obra
- 📦 **Controle de Materiais** - Organize e acompanhe materiais utilizados
- 📈 **Relatórios e Análises** - Visualize gráficos e dados em tempo real
- 💰 **Orçamentos Personalizados** - Crie orçamentos customizados
- 📱 **Funciona Offline** - Use sem conexão com a internet
- ⚡ **Acesso Rápido** - Ícone na tela inicial do seu celular

## 🚀 Como Usar

### 1. Instalar na Tela Inicial

**Android:**
- Abra o app no navegador
- Toque em ⋮ (menu)
- Selecione "Instalar aplicativo"

**iPhone:**
- Abra no Safari
- Toque em Compartilhar ↗️
- Selecione "Adicionar à tela inicial"

### 2. Acessar o App Completo

Após instalar, clique no botão "Abrir App" para acessar o gerenciador completo hospedado no Google AI Studio.

## 📱 Tecnologias Utilizadas

- **PWA (Progressive Web App)** - Funcionalidade de app nativo
- **Service Worker** - Suporte offline
- **Manifest.json** - Configurações do app
- **Google AI Studio** - Backend da aplicação

## 🔧 Configuração

O projeto está configurado para funcionar como PWA com:
- Cache offline
- Ícone customizado
- Notificações push (em desenvolvimento)
- Interface responsiva

## 🎨 Design

O design utiliza cores warm (laranja e vermelho) que remetem a construção e obra, com ícones representando:
- Martelo (ferramentas)
- Chave inglesa (manutenção)
- Gráfico de barras (gastos)
- Moeda (orçamento)
